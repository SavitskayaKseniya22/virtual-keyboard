# [Virtual keyboard](https://poetic-tarsier-ff7606.netlify.app/)

Simple virtual keyboard

## Tech Stack

- Javascript
- Sass
- Eslint
- Prettier
- Webpack
- Normalize.css

## How to use

1. Clone the repo

```bash
git clone https://github.com/SavitskayaKseniya22/virtual-keyboard.git
```

2. Change directory

```bash
cd virtual-keyboard
```

3. Install dependencies

```bash
npm install
```

4. Start the dev server

```bash
npm start
```
